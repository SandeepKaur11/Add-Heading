# Add Heading
This extension adds a heading to selected line.


Use any languages in the **Brackets 1.14**.

## How to use
Click the line you want to add a heading.


Both selected text and unselected text are acceptable.
#### Heading
**Edit > Add Heading**


Windows:  ⇧⌃-
<br>
Mac:  ⇧⌘-

#### SubHeading
**Edit > Add SubHeading**


Windows:  ⌥⌃-
<br>
Mac:  ⌥⌘-
